(16 pts) The pdf (probability density function) of \(X\) is given by
\[
f_{X}(x) = 
\begin{cases}
-\frac{1}{4}x, & -2 \le x < 0\\
cx^2, & 0 \le x \le 3\\
0, & \text{otherwise}
\end{cases}
\]
\begin{enumerate}[label=(\alph*)]
\item Find the value of the constant $c$.
\item Plot the pdf of $X$ from -3 to 4. If you plot by hand, make sure that your plot shows the
main features, i.e., where they intercept the y-axis and x-axis.
(c) Find the expression for the cdf (cumulative distribution function) of $X$.
(d) Find the expected value of $X$.
(e) Find the probability of the event $A = \{1 \le x \le 2\}$.
(f) Find the conditional pdf of $X$ given $A$, $f_{X|A}(x|A)$ and plot it from -3 to 4. If you plot by
hand, make sure that your plot shows the main features, i.e. where they intercept the y-
axis and x-axis.
(g) Find the conditional expected value of $X$ given $A$, $\mathbb{E}[X|A]$
\end{enumerate}
\paragraph{Solution}
\paragraph{(a)}
By the definition of pdf, we have
\[
\int_{-\infty}^{\infty}f_X(x)dx = 1
\]
and given the property of the function, we have
\[
\int_{-\infty}^{\infty}f_X(x)dx = \int_{-\infty}^{-2}0dx + \int_{-2}^{0}\left(-\frac{1}{4}x\right)dx + \int_{0}^{3}(cx^2)dx + \int_{4}^{\infty}0dx = 1 
\]
Which means
\[
(-\frac{1}{8}x^2)\bigg|_{-2}^{0} + (\frac{1}{3}cx^3)\bigg|_{0}^{3} = \frac{1}{2} + 9c = 1
\]
\(\Rightarrow \boxed{c = \frac{1}{18}}\)
